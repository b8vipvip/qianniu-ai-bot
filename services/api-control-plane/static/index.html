安全建议

  • 使用 Caddy/Nginx 和 HTTPS，对外只开放 80/443。
  • 修改默认管理员密码和 APP_SECRET。
  • 固定 API_KEY_ENCRYPTION_KEY，迁移服务器时一并备份。
  • 为每台 Bot 创建独立令牌，停用丢失设备的令牌。
  • 定期备份 /data/api-control-plane.db。
